LocalStorage Class

Defined in: src\saveManager-1.0.1.js:221
Module: Plugins

Handles the saving of information through localStorage for a Kiwi Game. All of the information that is saved using this object is stored based on the name of the Game. So be careful when dealing with multiple games of the same name, as the data can conflict.
